UnoCSS installieren!

cd ordner
npm create vite@latest . -- --template react-ts
npm install -D unocss
code .
npm run dev


1. Hast du UnoCSS in vite.config.ts eingebunden?

// vite.config.ts
import UnoCSS from 'unocss/vite'
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'

export default defineConfig({
  plugins: [
    react(),
    UnoCSS(), // << MUSS da rein!
  ]
})


2. Fehlt der Import von index.css in main.tsx?
// src/main.tsx
import React from 'react'
import ReactDOM from 'react-dom/client'
import App from './App.tsx'
import './index.css' // << wichtig!
import 'uno.css' // << das auch, wenn du kein CLI-Build nutzt

ReactDOM.createRoot(document.getElementById('root')!).render(
  <React.StrictMode>
    <App />
  </React.StrictMode>
)

3. Ist uno.config.ts korrekt konfiguriert?
// uno.config.ts
import { defineConfig } from 'unocss'
import { presetWind3 } from '@unocss/preset-wind3'

export default defineConfig({
  presets: [
    presetWind3(),
  ]
})

4. Index CSS 
/* src/index.css */
@import '@unocss/reset/tailwind.css';


5. Fehlt ein UnoCSS Build/Dev Step?
npm run dev

6. (Optional) Sicherstellen, dass Vite JSX nicht cached
rm -rf node_modules .vite dist
npm install
npm run dev
